# homebrew-extra
Missing formulae and casks for homebrew, primarily for my dotfiles bootstrap

# Use
```bash
brew [cask] install jrussellsmyth/extra/[cask|formula]
```

or

```bash
brew tap jrussellsmyth/extra

brew [cask] install [cask|formula]
```
